
## <a name="features">🔋 Features</a>

👉 Api fetching with efficient method and best practices.

👉 React Scroll for product listing.

👉 Store cart data in device, useful for native apps.

👉 Responsive across various devices, emphasizing Tailwind's mobile-friendly capabilities.

## ⚙️ Tech Stack

- TypeScript
- React
- Redux-Toolkit
- RTK Query
- Redux-persist

## <a name="quick-start">🤸 Quick Start</a>

Follow these steps to set up the project locally on your machine.

**Prerequisites**

Make sure you have the following installed on your machine:

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/en)
- [npm](https://www.npmjs.com/) (Node Package Manager)

**Cloning the Repository**

```bash
git clone https://github.com/noushadzaman/pixfar-shop
cd pixfar-shop
```

**Installation**

Install the project dependencies using npm:

```bash
npm install
```


**Running the Project**

```bash
npm start
```

Open [http://localhost:5173](http://localhost:5173) in your browser to view the project.
